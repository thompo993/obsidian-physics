This is a  informational place where the purpose of each subfolder is explained. 

### attachments: 
A disorganised place for all media, access is provided through a linked file in the main file dialogue. They are all stored here to prevent clutter, it could be files, reading material, lecture pdfs images, etc. 

### Tags: 
Tags are a place for all text files that do not have a clear place in the file dialogue, could be made of definitions, asides anything. Its a catch all for anything that doesn't have an obvious place in the main file dialogue. 

Files can be moved from here into the file dialogue when a place for them becomes clear. 

This file should never be navigated, they will be linked to from a different file in the main file dialogue. Consider the tag "MuSR" it is a definition, very relevant but no clear place to live, so it goes into tags so that it can be linked back to, and providing both a definition and  better connections across the zettelkasten. (MuSR not linked above as this readme doesn't need to be linked to anything.)

### Templates
This is where various templates for different document styles live.
